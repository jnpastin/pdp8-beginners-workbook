# PDP-8 Programming Workbook

A structured, progressive workbook for learning to write real PDP-8 programs in PAL8 assembly — from first principles through OS/8 system programming.

Built around a PiDP8/I running OS/8. The emulator and documentation are at the [TangentSoft repository](https://github.com/tangentsoft/pidp8i).

---

## Structure

Each chapter is a self-contained directory containing:
- **README.md** — goals, exercise descriptions, implementation rationale, and key learning points
- **EXn.PA** — PAL8 source files
- **EXn.LS** — assembler listing files

The full exercise plan, goals, and progress tracking are in [WoorkBook.md](WoorkBook.md).  
Style rules, idioms, and code review standards are in [Style_and_Conventions.md](Style_and_Conventions.md).
